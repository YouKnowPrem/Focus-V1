📌 What is Focus Time?

Focus Time is a productivity-based Android application designed to help users stay focused, track sessions, and analyze their productivity trends over time. It works completely offline using a local database for secure and fast data handling.

✨ Key Features

  • ✔ Offline local database (No internet required)
  • ✔ Productivity analysis & session tracking
  • ✔ Clean and smooth Flutter UI
  • ✔ Lightweight & optimized performance
  • ✔ Three focus modes (Currently under testing)
  • ✔ Designed to be stable and crash-free*

*Tested personally for stability. If you experience any crashes or bugs, please report them so they can be fixed in Version 2.
